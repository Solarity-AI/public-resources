This repository is the **single source of truth for all public-facing resources** of Solarity AI LLC.

## Purpose

The purpose of this repository is to:

- Centralize all public-facing company materials
- Prevent outdated, inconsistent, or unofficial assets from being shared
- Enable fast, safe distribution of approved resources
- Maintain versioned history and accountability for changes
- Serve as a canonical reference for internal teams

If a resource is public-facing and official, it belongs here.

---

## What Belongs in This Repository

Included:
- Pitch decks and investor materials
- One-pagers and company overviews
- Job descriptions and hiring collateral
- Brand assets and visual identity files
- Public legal, compliance, and policy documents
- Press kits and media resources
- Public demo assets and documentation

Not included:
- Source code or internal tooling
- Confidential or NDA-protected documents
- Drafts or unapproved materials
- Internal strategy documents
- Financials not explicitly marked as public

## Approval & Governance

- Only approved materials may be added or modified
- Every asset must reflect current company positioning and branding
- Changes should be reviewed before distribution
- This repository is not a sandbox

If you are unsure whether something belongs here, it probably does not.
